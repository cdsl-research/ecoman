# ECoMan - Easy Computer Manager

## How to run

Setup venv

```
python3 -m venv env
. env/bin/activate
pip installl -r requirements.txt
```

Starting app

```
python app.py
```

Acceess Web UI

```
http://<your-hostname>:3000/
```
